Configuration
=============

The following settings are available in the module configuration page that is
displayed right after enabling it

NAAN
----

The Name Assigning Authority Number of your organization. If you don't have one
yet, read `Getting started: what to plan for as you implement ARKs
<https://arks.org/about/getting-started-implementing-arks/>`_.

Shoulder
--------

The shoulder that will be used for all generated ARK identifiers. See `ARK
namespaces and sub-namespaces <https://arks.org/about/ark-namespaces/>`_ for
more informations about shoulders.
